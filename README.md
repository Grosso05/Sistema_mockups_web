# Fecha de lanzamiento inicial: 10 de enero de 2024, 4:50 pm

Este es el primer avance del proyecto, diseñado para generar un catálogo con productos personalizados mediante la inserción de un logo. En esta versión inicial, el programa solo genera un PDF en formato binario. Se espera que futuras versiones incluyan mejoras adicionales y nuevas características.
